# Aave Health Hook

An Uniswap v4 hook project for integrating Aave-based health checks into pool activity.

This repository contains the hook implementation, deployment scripts, test scaffolding, and generated artifacts used to build and verify the project.

## What’s Included

- `src/` - core hook contracts and interfaces
- `script/` - deployment and workflow scripts
- `test/` - Foundry tests and test utilities
- `lib/` - vendored dependencies and helper libraries

## Getting Started

Build and test with Foundry:

```bash
forge build
forge test
```

## Notes

The repository is structured for Solidity development with Foundry and includes generated build outputs and cached deployment data from local development.
